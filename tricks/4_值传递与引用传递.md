### 值传递 pass by value
```text
void change(int x){
    x = 100;
}

//主函数
int a = 10;
change(a);

cout << a;
```
调用函数时：a = 10  
系统偷偷复制一个：x = 10

### 引用传递
```text
void change(int& x){
    x = 100;
}

//是同一块内存！
a -----> 10
x -----↑

```

### & 是“取地址”，* 是“通过地址拿值（解引用）”

#### 定义时：
```text
int* p;
int& r = a;
* → 指针
& → 引用
```
#### 使用时：
```text
&a
*p
& → 取地址
* → 根据地址拿值
```
