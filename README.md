# cpp-algo-handbook

A compact C++ algorithm handbook for STL usage, data structure templates, common algorithms, and coding tricks.

## Structure

- [STL](./STL/) - common container operations and usage notes
- [data_structure](./data_structure/) - implementation templates for classic data structures
- [algorithms](./algorithms/) - core algorithm templates
- [tricks](./tricks/) - practical tricks and examples
- [leetcode_notes](./leetcode_notes/) - reserved for problem notes

## How to use

- Read the STL notes when you want quick operations and syntax.
- Copy the implementation templates from `data_structure` and `algorithms` into your own solutions.
- Keep small bug reminders and trick snippets in `tricks`.
