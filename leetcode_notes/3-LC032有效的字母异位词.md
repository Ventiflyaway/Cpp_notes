### C++ 11 范围for循环
```text
// 正确写法：用冒号，不是用括号和冒号
for (char c : str) {
    std::cout << c << std::endl;
}

// 如果需要修改字符，用引用
for (char& c : str) {
    c = toupper(c);
}
```

### 方法1：一个 hashmap，边加边减

初始做法：（内存占用9.71MB）
- 建两个 hashmap
- 再比较

其实一个 map 就够了：
- s 的字符：+1
- t 的字符：-1
- 最后检查是不是全 0

### 方法2：数组代替 hashmap（更强，有助于系统！！）

在实际工程和面试里：已知 key 范围固定 → 优先数组，因为int cnt[26]是连续内存，cache friendly。
```text
for(int i = 0; i < s.length(); i++) {
    cnt[s[i] - 'a']++;
    cnt[t[i] - 'a']--;
}
```
s[i] - 'a'：字符相减，实际是 ASCII 码相减
结果是一个 0~25 的整数，对应字母表中的索引
（内存：9.31MB）

### 方法3：排序法
```text
class Solution {
public:
    bool isAnagram(string s, string t) {
        sort(s.begin(), s.end());
        sort(t.begin(), t.end());

        return s == t;
    }
};
```
