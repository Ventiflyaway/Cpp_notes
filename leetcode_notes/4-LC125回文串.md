字母和数字都属于字母数字字符

用到的函数：
- isalnum(c): 是不是字母或数字 
  - 0：不是字母或数字
```text
for (char c : s) {
    if (isalnum(c)) {
        filtered += c;  // 只保留字母和数字
    }
}
```

### 字符串size/length
- length() 更符合“字符串长度”的语义
- size() 是为了和 STL 容器统一

建议用size():写泛型、STL 时，习惯会更统一

有一个 C++ 新手容易踩的坑:空字符串
int right = s.size() - 1;
int right = (int)s.size() - 1;
s.size() 返回的是无符号整数（unsigned）
