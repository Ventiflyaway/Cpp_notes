### 考察点：指针
- 要求：输入头节点指针，逆转链表，输出逆转链表的头节点指针
### 关键：3个对象prev,cur,next
指针改方向时，需要保证前后不丢失
- 保存next指针
- 把视角一直往右移动（直到当前是nullptr），每次都让next指针指向前一个元素

```text
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode * prev = nullptr;
        ListNode * cur = head;
        while(cur != nullptr){
            ListNode *next = cur->next;
            
            cur->next = prev;

            prev = cur;//当前节点已经处理完了，保存一下，存成prev
            cur = next;//整体前进一个节点，处理下一个节点
        }
        return prev;
    }
};
```
