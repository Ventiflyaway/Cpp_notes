### 要求：O(1) 找到最小值
则：不能遍历

### 思路：两个栈，第二个栈压入最小值（前 i 个元素的最小值）
- 时间复杂度 所有操作：O(1)
  - push、pop、top、getMin 都只做了 固定次数操作
- 空间复杂度：O(n)

### 构造函数 = 对象出生时自动执行的函数。
- 作用：初始化对象，给它初始状态（这样就不会是垃圾值）
- 是C++ OOP（面向对象）的核心概念之一！
```text
class Student {
```

```text
# include <stack>

class MinStack {
public:
    /** initialize your data structure here. */
    stack<int> st;
    stack<int> minst;
    MinStack() {
        //这个就是构造函数
    }
    
    void push(int x) {
        st.push(x);
        if (minst.empty()) minst.push(x);
        else{
            int minINT = min(x,minst.top());
            minst.push(minINT);
        }
        
    }
    
    void pop() {
        st.pop();
        minst.pop();
    }
    
    int top() {
        return st.top();
    }
    
    int getMin() {
        return minst.top();
    }
};

/**
 * Your MinStack object will be instantiated and called as such:
 * MinStack* obj = new MinStack();
 * obj->push(x);
 * obj->pop();
 * int param_3 = obj->top();
 * int param_4 = obj->getMin();
 */
```
